#Glow Haven Beauty Studio Website

  ##Student Information
  Name: Karabo Mojalefa
  Student Number: ST10516068

  ##Project Overview
  This project is a website for a beauty salon. It provides information about services, showcases images, and allows users to book appointments.

  ##Website Goals and Objectives
  -Provide clear and detailed information about services
  -Attract customers through pictures
  -Allow online booking
  
  ##Key Features and Functionality
  -Navigation menu (Home, Contact, Services, etc.)
  -Image gallery (Images used in this project were sourced from Unsplash and Pixels, which provide royalty-free images for educational use.)
  -Booking form
  -Contact page

  ##Timeline and Milestones
  -Week 2: Planning and proposal
  -Week 2: HTML development
  -Week 3: Testing and finalisation

  ##PART_1 Details
  This phase focuses on building the website using HTML only, without styling.
